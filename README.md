## 📌 About the Project

This is a simple C++ console application that allows the user to register either a client or an employee and display their information.

---

## 🚀 Features

- 👤 Register clients and employees
- 🧱 Object-Oriented Programming
- 🔁 Conditional logic system
- 🖥️ Console-based interaction

---

## 🛠️ Technologies

- C++
- Standard Library (`iostream`, `string`)

---

## 💻 How it works

1. User selects:
   - 1 → Client
   - 2 → Employee
2. Inputs personal data
3. Program displays formatted output
